---
title: Clearinghouse State
description: Get positions, margin summary, and account value for a user
---

<Note>
  `POST /info`
</Note>

Returns the complete trading state for a user: open positions, margin usage, account value, and withdrawable balance.

## Request body

```json
{
  "type": "clearinghouseState",
  "user": "0xYourAddress"
}
```

<ParamField body="user" type="string" required>
  Wallet address.
</ParamField>

### Position fields

| Field | Type | Description |
|-------|------|-------------|
| `coin` | `string` | Asset name |
| `szi` | `string` | Signed size (positive = long, negative = short) |
| `entryPx` | `string` | Weighted average entry price |
| `positionValue` | `string` | `|szi| × markPx` |
| `unrealizedPnl` | `string` | Current unrealized profit/loss |
| `returnOnEquity` | `string` | `unrealizedPnl / marginUsed` |
| `liquidationPx` | `string \| null` | Estimated liquidation price |
| `leverage` | `object` | `{type: "cross"|"isolated", value: number}` |
| `marginUsed` | `string` | Margin consumed by this position |

### Margin summary fields

| Field | Type | Description |
|-------|------|-------------|
| `accountValue` | `string` | `balance + totalUnrealizedPnl` |
| `totalNtlPos` | `string` | Total notional position value |
| `totalRawUsd` | `string` | Balance (without PnL) |
| `totalMarginUsed` | `string` | Sum of all position margins |

## Example

```bash
curl -s http://localhost:3000/info \
  -H 'Content-Type: application/json' \
  -d '{"type": "clearinghouseState", "user": "0xtest"}'
```
